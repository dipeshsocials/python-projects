Number Guessing Game

A command-line game where the player guesses a randomly generated number between 1 and 100.

Features

- Random number generation
- Unlimited guesses
- Input validation
- Attempt counter
- Helpful hints ("Too high" / "Too low")

Concepts Practiced

- "random" module
- "while" loops
- "try" / "except"
- Conditional statements
- User input

Run

python guessing_game.py
