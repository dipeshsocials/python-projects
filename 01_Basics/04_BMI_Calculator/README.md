BMI Calculator

A simple Python command-line application that calculates Body Mass Index (BMI) and displays the corresponding weight category.

Features

- Calculates BMI
- Displays BMI up to two decimal places
- Categorizes BMI:
  - Underweight
  - Normal Weight
  - Overweight
  - Obesity
- Handles invalid input

Concepts Practiced

- Functions
- Conditional statements
- Floating-point arithmetic
- Exception handling

Run

python bmi_calculator.py
