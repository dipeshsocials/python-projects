Palindrome Checker

A Python program that checks whether a word, phrase, or sentence is a palindrome.

Features

- Ignores spaces and punctuation
- Case-insensitive comparison
- Works with words and sentences

Examples

- "madam" → Palindrome
- "racecar" → Palindrome
- "A man, a plan, a canal: Panama" → Palindrome
- "python" → Not a palindrome

Concepts Practiced

- Functions
- String manipulation
- Slicing
- Loops
- Boolean logic

Run

python palindrome.py
