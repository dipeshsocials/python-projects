Armstrong Number Checker

A Python application that checks whether a number is an Armstrong number and can also list all Armstrong numbers within a specified range.

Features

- Check a single number
- Find Armstrong numbers in a range
- Input validation
- Clean function-based implementation

Example

Choose an option: 1
Enter a number: 153

Output:
153 is an Armstrong Number.

Concepts Practiced

- Functions
- Loops
- String manipulation
- Mathematical operations
- Range iteration

Run

python armstrong.py
