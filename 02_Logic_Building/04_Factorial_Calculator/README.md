Factorial Calculator

A Python program that calculates the factorial of a number using both iterative and recursive approaches.

Features

- Iterative solution
- Recursive solution
- Input validation
- Easy comparison of both methods

Example

Input:
5

Output:
Iterative Result : 120
Recursive Result : 120

Concepts Practiced

- Functions
- Loops
- Recursion
- Input validation

Time Complexity

- Iterative: O(n)
- Recursive: O(n)

Run

python factorial.py
